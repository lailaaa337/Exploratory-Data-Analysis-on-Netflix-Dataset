# Netflix Titles Data Analysis

## Overview

This project focuses on analyzing a dataset of Netflix movies and TV shows to understand patterns in content available on the platform. The goal was to explore the data and extract useful insights.

## What I did

* Cleaned the dataset and handled missing values
* Removed unnecessary columns
* Renamed columns for better clarity
* Processed dates and durations
* Performed exploratory data analysis

## Key insights

The analysis shows differences between movies and TV shows, content distribution across countries, and how genres and durations vary. It also highlights trends in when content was added to Netflix.

## Tools used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn

## Notes

This project helped me practice data cleaning and exploratory data analysis, which are essential steps in any data science workflow.
